Quick Reference

Essential tips for daily Business Central work

Note: This section provides quick reminders of key concepts. For detailed explanations, see the Appendix.

Essential Keyboard Shortcuts

Shortcut Action
Alt+Q Open "Tell Me" search
Alt+F3 Filter to current field value
Shift+F3 Open filter pane
Ctrl+Shift+F12 Toggle focus mode on lines
Ctrl+C / Ctrl+V Copy and paste rows
Shift+Enter Quick Entry reverse direction
F5 Refresh page

Quick Data Entry Tips

  • Dates: Type just the day (12) for current month/year, or day+month (1205) for current year
  • Calculations: Enter formulas directly in number fields (10+10, 2*6, 30-15, 12/6)
  • Quick Entry: Use Enter key to jump through only the fields you care about (customize with personalization)
  • Copy Multiple Rows: Ctrl+Click or Shift+Click to select, then Ctrl+C to copy

Quick Search and Filter Tips

  • Search across all fields: Use the search icon at the top of any list
  • Filter a single field: Alt+F3 on a selected value
  • Advanced filtering: Use the filter pane (Shift+F3) with symbols:
    • .. for ranges (1000..2000)
    • | for "or" (10|25)
    • <> for "not equal to" (<>0)
    • * for wildcards (*wine*, wine*)
    • @ for case-insensitive (@wine*)

Tell Me Feature

The Tell Me search (Alt+Q or click the lightbulb icon) helps you find:

  • Actions on the current page: Tasks you can perform on the open page
  • Pages and tasks: Navigate to lists, cards, and setup pages
  • Reports and analysis: Access reporting tools
  • Documentation: Find help articles
Pro Tip

Tell Me uses synonyms. Searching for "products" will find "items."
