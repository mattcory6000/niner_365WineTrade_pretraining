Understanding What You'll Learn

Important Note

The learning modules in this guide use Microsoft's Cronus demonstration company. This is standard Business Central without industry-specific customization.

The 365WineTrade version of Business Central you'll use in production includes significant enhancements for wine production and sales that you won't see in these training exercises. We've noted key differences throughout this guide.
