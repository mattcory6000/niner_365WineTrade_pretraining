Admin & Leadership Role Track

Master user management, security, email setup, and data analysis tools. These modules are essential for system administrators, IT managers, and business leaders responsible for maintaining and optimizing Business Central.

Before You Start: Complete Rounds I-III first to build your foundation in Business Central basics. Admin tasks require a solid understanding of how the system works from a user perspective.

  • Creating and saving custom views
  • Filtering and grouping data
  • Exporting data to Excel
Start Module
Wine Industry Data Analysis

Business Central's analysis tools are powerful for wine data:

  • Inventory analysis: Group by vintage, producer, or region to see stock levels
  • Sales trends: Analyze sales by varietal, price point, or customer segment
  • Customer insights: Identify top customers by wine category or allocation tier
  • Allocation tracking: Monitor allocated vs. available quantities across your portfolio

365WineTrade Permission Sets

365WineTrade includes pre-configured permission sets for wine industry roles:

  • Wine Sales Rep: Access to sales orders, customer allocations, and pricing
  • Warehouse Manager: Full access to inventory, bins, and shipping
  • Finance Manager: GL access, reconciliation, and reporting
  • Compliance Officer: View-only access to compliance reports and TTB data

Tip: Start with these base permission sets and customize as needed for your organization.

Admin & Leadership Skills Checklist

After completing this track, you should be able to:

  • Use Analysis Mode to explore and analyze data across any list in Business Central
  • Create and manage user accounts with appropriate permissions
  • Assign and customize permission sets for different roles
  • Configure email accounts and document sending profiles
  • Set up Outlook integration for seamless workflow
  • Monitor user activity and system usage
  • Export data for further analysis in Excel or Power BI
  • Troubleshoot common user access and email issues
