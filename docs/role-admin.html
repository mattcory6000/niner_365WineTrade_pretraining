Admin & Leadership Role Track

Master user management, security, email setup, and data analysis tools. These modules are essential for system administrators, IT managers, and business leaders responsible for maintaining and optimizing Business Central.

Before You Start: Complete Rounds I-III first to build your foundation in Business Central basics. Admin tasks require a solid understanding of how the system works from a user perspective.

In 365WineTrade, you'll have access to specialized wine industry analysis tools:

üìù 365WineTrade Notes: Wine Industry Security

365WineTrade adds specialized permission controls for wine operations:

  • Approval User Setup - designate who can approve Sales Order Reviews vs. Credit Reviews
  • Document Group Permissions - control access to specific document types (COLAs, contracts, etc.)
  • Internal Permission System - granular control over wine-specific features beyond standard BC permissions
  • Order Review Categories - define which types of orders require approval and by whom

These complement Business Central's standard security but add wine industry-specific controls.

üìù 365WineTrade Notes: Advanced Document Distribution

365WineTrade includes powerful Document Worksheets for batch document processing:

  • Email Report Setup - configure which BC reports can be sent automatically
  • Document Worksheet Templates - create templates for order confirmations, invoices, statements, etc.
  • Email Bodies - pre-built email templates with variables
  • Batch Processing - send hundreds of documents (invoices, confirmations) to customers/vendors at once
  • Email Addresses by Document Type - customers can have different emails for invoices vs. statements

This goes far beyond standard BC email capabilities and is essential for high-volume wine distribution.

Admin & Leadership Skills Checklist

After completing this track, you should be able to:

  • Use Analysis Mode to explore and analyze data across any list in Business Central
  • Create and manage user accounts with appropriate permissions
  • Assign and customize permission sets for different roles
  • Configure email accounts and document sending profiles
  • Set up Outlook integration for seamless workflow
  • Monitor user activity and system usage
  • Export data for further analysis in Excel or Power BI
  • Troubleshoot common user access and email issues
üìù 365WineTrade Notes: What's Next for Admins

When you transition to 365WineTrade, admins will need to configure:

  • Wine Setup - the master configuration page for all 365WineTrade features
  • Order Review Categories - define approval workflows for credit holds, sample orders, pricing changes
  • Chargeback Setup - configure Depletion Allowance, Bill Back, and Bad Bottle processing
  • Allocation Setup - define how inventory allocations work (if using this feature)
  • State License Setup - configure compliance requirements by state
  • Integration Sites - set up connections to State Liquor Authorities or other external systems

Your implementation team will guide you through these during your 365WineTrade training.
